# Plantilla LaTeX para Proyectos Finales - DCT
### Departamento de Ciencias de la Tecnología e Innovación
### Universidad Católica Boliviana "San Pablo"

Esta plantilla oficial ha sido desarrollada para estandarizar la documentación técnica de los proyectos finales de semestre en las carreras de Ingeniería de Sistemas, Mecatrónica y Biomédica.

---

## 1. Instrucciones de Uso

### Requisitos
* En la nube: Compatible con Overleaf (recomendado).
* Local: Requiere una distribución de LaTeX (MiKTeX o TeX Live) y un editor como VS Code o TeXstudio.

### Configuración Inicial
1. Descargue el contenido del repositorio.
2. Edite únicamente el archivo proyecto.tex para el contenido de su trabajo.
3. No modifique el archivo catolica_proyectos.cls, ya que contiene la estructura institucional.

---

## 2. Estructura del Proyecto

* catolica_proyectos.cls: Archivo de clase con la lógica de diseño y carátula.
* proyecto.tex: Documento raíz del proyecto.
* Figuras/: Directorio para el almacenamiento de imágenes y diagramas.

---

## 3. Personalización

En el preámbulo de proyecto.tex debe completar los siguientes comandos para generar la carátula:

- \titulo{Nombre del Proyecto}
- \carrera{Nombre de la Carrera}
- \materia{Nombre de la Materia}
- \autor{Nombre del Estudiante}
- \docente{Nombre del Docente}
- \semestre{Semestre/Año}
